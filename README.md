# film-store-backend

## Python Virtual Environment

### Activate using Python:
``` 
source venv/bin/activate
python app.py
```
### Activate using Flask CLI:
``` 
source venv/bin/activate
export FLASK_DEBUG=1 
flask run --port=3001  
```
### Deactivate:
```
deactivate 
```
